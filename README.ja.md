# echizen-city-web

越前市のWebサイトのリポジトリです。

## デモ

- https://code4fukui.github.io/echizen-city-web/static/demo1/
- https://code4fukui.github.io/echizen-city-web/static/demo2/

## 機能

- 市民向け・観光客向けのページを提供
- 便利なメニュー、市政情報、広報・報道などの情報を掲載
- 講座・教室のコースや入園・入学、引っ越し、結婚・離婚などの情報を提供

## 必要環境

特になし。

## 使い方

リポジトリをクローンし、static以下のページを確認できます。

## データ・API

外部のデータやAPIは使用していません。コンテンツはすべて静的なものです。

## ライセンス

特に定めていません。
